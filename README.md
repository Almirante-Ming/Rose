aplicativo de gestao de aulas com foco para gestao de atividades por usuario vinculando um terceiro como mediador, seu uso foi pensado para clinicas de fisioterapia, academia e outros estabelecimentos com atividades similares.
* o aplicativo nao possui uma pagina de registro, pois isso deve ser feito pelo estabelecimento no qual deseja vincular, uma vez feito, o cliente consegue acessar e gerenciar normalmente suas atividades.
* ha a opcao de utilizar o servidor self hosted[https://github.com/Almirante-Ming/tinto] para maior privacidade, pensando nisso o aplicativo ja esta equipado com a opcao de indicar o servidor logo na tela de login.

ha planos futuros para uma integracao via whatsapp com um painel web para gerenciamento por parte do estabelecimento, por enquanto tanto o estabelecimento quanto instrutores e alunos podem usar para acompanhar as atividades agendadas bem como interagir com coisas basicas como remarcar ou cancelar.
